## Example API authentication for user

A simple application which is demonstrate:

-   migration for User
-   factory for User
-   seeder for User
-   api routes
-   UserController
-   UserResource
-   test for User

If you want to test application, please do next operations in the terminal:

```
  make prepare-test
  make test
```
